# MIND

A full-stack monorepo for AI-powered systems with web, mobile, bot, and infrastructure components.

## Project Structure

- **apps/web** - Next.js web application
- **apps/mobile** - React Native (Expo) mobile app
- **apps/bot** - Node.js bot service
- **packages/shared** - Shared types and database utilities
- **packages/ui** - Reusable UI components
- **packages/engine** - Core business logic engine
- **packages/canvas** - Canvas/visualization library
- **infra/docker** - Docker compose configuration

## Quick Start

```bash
pnpm install
pnpm dev
```

## Technologies

- Next.js 15
- React 18
- React Native / Expo
- TypeScript
- Tailwind CSS
- PostgreSQL / TimescaleDB
- Clerk Authentication
- Liveblocks Collaboration
